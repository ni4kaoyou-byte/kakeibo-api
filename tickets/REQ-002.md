# REQ-002 MVP機能の確定

- Status: Done
- Priority: Must
- Depends On: REQ-001

## Goal

初期リリースに含める機能を確定する。

## Scope

### Must (MVP)

- 認証（最低限、ユーザー単位でデータ分離できること）
- 取引CRUD（収入/支出、日付、金額、カテゴリ、メモ）
- カテゴリCRUD
- 日次・月次・年次集計API（収入合計、支出合計、収支）
- 基本バリデーションとエラーレスポンス統一

### Should (MVP直後)

- 月次カテゴリ別集計
- 期間/カテゴリ/収支種別の検索・フィルタ
- ページング

### Could (後続)

- 予算管理
- 定期取引
- CSV入出力
- 通知/アラート

## Out of MVP

- 予算管理
- 定期取引
- CSV入出力
- 通知/アラート

## Acceptance Criteria

- [x] Must/Should/Could に分類されている
- [x] MVPに含める機能と除外機能が明確
- [x] REQ-001の目的に直結する機能のみが Must にある
